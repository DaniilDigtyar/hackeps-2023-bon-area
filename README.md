# hackeps-2023-bon-area
Hackeps 2023 - Repte Bon Area

## Requirements
- python > 3.11

## Create venv
```
python -m venv venv
```

## Activate venv
### Windows
```
# In cmd.exe
venv\Scripts\activate.bat
# In PowerShell
venv\Scripts\Activate.ps1
```

### Linux
```
$ source myvenv/bin/activate
```

### Install packages
```
pip install -r requirements.txt 
```
